# 🇿🇲 Zambia Plant Disease Detector

## Identify Plants & Diseases Affecting Zambian Crops

An AI-powered web application that helps Zambian farmers identify crop diseases using their smartphone camera. **Now with plant identification and local disease names in Chichewa!**

## 🌟 Features

- ✅ **Identifies BOTH Plants and Diseases**
- ✅ **Zambian Localization** - Disease names in Chichewa
- ✅ **Zambian Crops Database** - Maize, Soybeans, Groundnuts, Cassava, Sunflower, Cotton, Tomatoes, Cabbage
- ✅ **Zambia-Specific Advice** - Treatment recommendations for Zambia's climate
- ✅ **Back Camera Support**
- ✅ **Bilingual Interface** (English & Chichewa)
- ✅ **Offline Capable**
- ✅ **100% Free**

## 🇿🇲 Supported Zambian Crops

| Crop | Local Name | Common Diseases |
|------|------------|-----------------|
| Maize | Chimanga | MLN, Gray Leaf Spot |
| Soybeans | Soya | Soybean Rust |
| Groundnuts | Njugu | Early Leaf Spot |
| Cassava | Manioc | Cassava Mosaic Disease |
| Sunflower | Mpendadzuwa | Sunflower Rust |
| Cotton | Thonje | Cotton Leaf Curl |
| Tomatoes | Matimati | Late Blight |
| Cabbage | Kabichi | Black Rot |

## 📱 How to Use

1. **Open the app** on your smartphone
2. **Click "Yambitsa Kamera"** - Allow camera access
3. **Take a clear photo** of the affected plant leaf
4. **Click "Jambulani & Zindikirani"**
5. **Get diagnosis** in English and Chichewa with treatment advice
